# Project Constitution: My Brain (Agentic OS)

## Core Principles

### 1. Frictionless Capture
Every thought, idea, decision, or resource must be storable in under 10 seconds. The cost of capturing must approach zero, or the system fails.

### 2. Text-First, Tool-Augmented
All knowledge lives in plain Markdown. Tools, automations, and agents wrap around the text, not the other way around. No lock-in — the vault is readable with any text editor.

### 3. Progressive Disclosure
Show the minimum needed for the current context. The home dashboard shows today — not every project. A project hub shows that project — not every task. Agent status shows what's running and what needs decisions — not every log line. Drill down as needed.

### 4. Compose, Don't Duplicate
Skills compose into routines. Routines compose into workflows. Agents compose skills into plans. A skill defined once is linked from everywhere it's used — never copied.

### 5. Local-First, Sync-Second
The system works fully offline on the primary machine. Agent inference that requires remote APIs degrades gracefully when offline (queues tasks, executes on reconnect). Sync to mobile or cloud is a convenience, not a dependency.

### 6. Agentic Orchestration with Human Guardrails
AI agents plan and execute autonomously within their capability boundary. They pause for human judgment at three gates: creative choices (which option?), destructive actions (publish? delete? spend?), and ambiguous situations (unclear input, unexpected error). The human is the executive; agents are the executors.

### 7. Continuous Evolution
Templates, skills, and agent capability definitions are living documents. Every execution — human or agent — is an opportunity to improve the template. The system gets better with use.
