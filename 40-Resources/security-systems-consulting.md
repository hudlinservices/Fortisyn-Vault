# Low-Voltage & Security Systems Consulting

The technical domain Roy Hudlin has practiced for 35 years and the discipline that connects the vault's "work" cluster: [[10-Projects/royhudlin/professional-history|Roy's career]], the [[10-Projects/aesengineering/_corporate-hub|AES Engineering]] client engagement, and [[10-Projects/innovatience/_corporate-hub|Innovatience]]'s consulting revenue. This page defines the field so those pages can reference a shared concept instead of repeating acronyms.

## What It Is

Low-voltage and electronic security systems consulting is the design, specification, and project management of the **non-power building systems** that run a modern facility: communications cabling, networks, access control, video surveillance, fire/life-safety, and detection. Consultants don't install — they **design the systems and write the specifications** that contractors bid and build to. Roy's roles span the full arc: technician → designer/PM → estimator → senior consultant → bid manager.

## The MasterFormat Backbone (Division 27 & 28)

In the CSI **MasterFormat** specification standard that governs North American construction documents, this work lives in two divisions:

| Division | Scope | Examples |
|----------|-------|----------|
| **Division 27 — Communications** | Telecommunications infrastructure: the non-active (non-powered) physical layer | Copper & fiber cabling, equipment racks, ladder rack, cable trays, conduit, horizontal cabling, network switches/routers/firewalls |
| **Division 28 — Electronic Safety & Security** | Active security and life-safety systems | Access control, video surveillance, intrusion detection, fire alarm, duress |

A modern reform groups the **cabling/conduit/raceways for security systems into Division 27**, while Division 28 covers the application-specific equipment — so a competent consultant coordinates *across* both divisions. This is exactly the "specification writing (NMS/MasterFormat)" skill listed in [[10-Projects/royhudlin/professional-history|Roy's profile]].

## The Consulting Workflow

1. **Bid management & proposals** — Roy worked as Bid Manager (980 Howe St ACS Upgrade) and proposal writer, using **Procore** for bid coordination.
2. **Specification writing** — authoring the 3-part MasterFormat sections contractors build to (e.g. Cariboo Memorial Hospital Phase 2 security specs).
3. **Master planning** — co-authoring electrical engineering master plans (Wrinch Memorial Hospital).
4. **Lead consulting** — security + telecom lead on major hospital and P3 projects across BC and Vancouver Island.
