# Chart of Accounts Pro

Professional accounting application — Node.js project with dist, public, src, tests, and uploads directories.

## Structure

```
accounting/
├── dist/      ← Built output
├── public/    ← Static public assets
├── src/       ← Application source
├── tests/     ← Test suite
└── uploads/   ← User uploads
```

## What It Implements

Chart of Accounts Pro is the *tool* behind the vault's finance domain. The accounting and tax concepts it manages — the five-category chart of accounts, GST/HST reporting, the Canadian small-business-deduction rules, multi-entity consolidation across the 11 Fortisyn entities — are documented in **[[40-Resources/accounting-multi-entity-finance|Accounting, Bookkeeping & Multi-Entity Finance]]**. That page also threads Roy's actual finance questions (CRA passive income, GST/payroll setup, salary-vs-dividends, single-client corporation rules) and the [[40-Resources/costa-rica|Costa Rica]] dual-residency tax layer.
