# SLF for ROY

**Personal AI companion.** A Django + React Progressive Web App that functions as a tutor, coach, assistant, and mentor. The most personal project in the Python Slayers portfolio — built for Roy, by Python Slayers.

## Purpose

A single AI companion that serves four roles:
- **Tutor** — learn Spanish, explore new topics, build skills
- **Coach** — daily accountability, habit tracking, goal progression
- **Assistant** — task management, scheduling, information retrieval
- **Mentor** — guided reflection, decision frameworks, creative brainstorming

## Tech Stack

- **Backend:** Django API + knowledge graph
- **Frontend:** React PWA (offline-capable, mobile-first)
- **Vault:** Personal knowledge vault (structured, queryable)
