# Signatures

**Brand identity system for the Fortisyn Group.** Manages visual assets — logos, colors, typography, email signatures, and brand guidelines — across all 11 entities. A core Auron Media service delivered as a project.

## Brand Identity Components

| Component | Status | Details |
|-----------|--------|---------|
| **Logo Suite** | Planned | Primary, secondary, icon/favicon per entity |
| **Color Palette** | Planned | Hex, RGB, CMYK per entity |
| **Typography System** | Planned | Heading/body fonts, weights, type scale |
| **Email Signatures** | In Progress | Templates for entities and individuals |
| **Business Cards / Letterhead** | Planned | Print templates |
| **Social Media Templates** | Planned | Profile and cover graphics per platform |
| **Brand Guidelines** | Planned | Central PDF per entity |

## Design Principle

Maintain a consistent visual "family" across the Fortisyn parent company while allowing entity-specific customization. Each entity gets its own identity within the umbrella.
